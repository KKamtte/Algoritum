병합 정렬(merge sort) 알고리즘을 이용하여 도형을 면적 오름차순으로 정렬하는 자바 프로그램을 작성하시오.
도형 인터페이스, 원 클래스, 삼각형 클래스는 hw4_4와 동일함
5개의 클래스/인터페이스로 이루어진 프로그램임
** 앞으로 식별자 이름은 과제 설명을 그대로 따를 필요 없으며 자바 관례에 맞게 각자 알아서 정할 것
** 앞으로 수행시간 복잡도를 고려하여 알고리즘을 구현할 것

 

- 도형 인터페이스


- 원 클래스 - 도형 인터페이스를 구현


- 삼각형 클래스 - 도형 인터페이스를 구현

 

- 정렬 관련 기능을 모아놓은 클래스(예를 들어 FigureSortService)
mergeSort 메소드 - 도형 배열을 매개변수로 받는 static method
mergeSort 메소드 - 도형 배열, 정렬할 하한 인덱스, 상한 인덱스를 매개변수로 받는 static method
merge 메소드 - 도형 배열, 병합할 하한 인덱스, 상한 인덱스를 매개변수로 받는 static method

 

- 드라이버 클래스(예를 들어 MergeSortTest)
main 메소드는 다음을 수행

 

(1) 원 갯수(n1)를 입력받음
(2) 삼각형 갯수(n2)를 입력받음
(3) 크기가 n1+n2인 도형 배열을 생성함(예를 들어 figureArray)
(4) n1개의 원 정보(반지름)를 입력받아 원 객체를 생성하여 도형 배열에 저장
(5) n2개의 삼각형 정보(밑변과 높이)를 입력받아 삼각형 객체를 생성하여 도형 배열에 저장
   밑변과 높이 입력시, 쉼표 기호 등이 없이 whitespace로 구분하여 입력할 것
(6) 도형 배열을 병합 정렬
   예를 들어 다음과 같이 호출할 수 있다.
   FigureSortService.mergeSort(figureArray);
(7) 면적 순으로 정렬된 n1+n2개의 도형 정보와 면적을 출력

 

- 입력: 원 갯수(n1), 삼각형 갯수(n2), n1개의 원 정보, n2개의 삼각형 정보
- 출력: 면적 오름차순으로 정렬된 원과 삼각형 정보와 면적

 

- 실행 예:
hw4_5: 홍길동

 

원 갯수 입력: 3
삼각형 갯수 입력: 2


3개의 원 정보(반지름)를 입력하세요:
1.2
2.5
1.7


2개의 삼각형 정보(밑변과 높이)를 입력하세요:
4.5 4.5
2.3 1.9

 

병합 정렬 결과 5개의 도형 정보와 면적 =
삼각형 밑변=2.3 높이=1.9 면적=2.185
원 반지름=1.2 면적=4.5216
원 반지름=1.7 면적=9.0746
삼각형 밑변=4.5 높이=4.5 면적=10.125
원 반지름=2.5 면적=19.625
